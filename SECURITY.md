# Security Policy

## Supported Scope

This profile repository only contains public-facing profile content and static assets.

Do not open public issues for:
- account recovery or login problems
- private email or identity data
- secrets or tokens from unrelated repositories
- anything that requires revealing private personal information

## Reporting

Report security issues privately to `hello@arifaqyl.me`.

Include:
- affected file or asset
- reproduction steps
- impact
- whether the issue exposes personal or account data
